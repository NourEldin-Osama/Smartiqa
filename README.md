# Smartiqa Project

to install dependency
```commandline
pip install -r requirements.txt
```

to configure database
```commandline
python manage.py makemigrations
python manage.py migrate
python manage.py sqlmigrate Smart 0001
```

to create superuser
```commandline
python manage.py createsuperuser
```
